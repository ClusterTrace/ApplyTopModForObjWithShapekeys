This is a script created to be used within Blender, a free, mutlipurpose software.
This is a python script made to get around an annoying issue present in the software for applying modifiers to objects that use shapekeys.

To use:
1. Have a text editor window open in Blender
2. Drag an drop or open the script in the text editor
3. Select the object you want to use the script on with your cursor
4. Hit the run button in the text editor window
